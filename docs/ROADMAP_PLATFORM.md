# Platform Roadmap

Long-term direction for Atlas as a tool for transit planning and public accountability.

---

## Access & Equity

- [ ] **Walkshed modeling**: walking access maps generated from the GTFS stop graph — how far can you get from a stop in 5/10/15 minutes on foot
- [ ] **Demographic overlay**: Census income and minority population data layered against frequent transit coverage, for Title VI compliance analysis
- [ ] **Coverage scoring**: automated measure of how well high-frequency routes serve priority areas

---

## Network Analysis

- [ ] **Before/after service change**: compare frequency maps across two schedule periods to show what changed and where
- [ ] **Route redesign sandbox**: propose a new route or frequency change, see how it affects coverage without touching the live data

---

## Public Tools

- [ ] **Share a view**: shareable URL for a specific map state (agency, filter, date) — for advocacy, journalism, public consultation
- [ ] **Multi-agency merged view**: combine adjacent regional agencies into a single continuous network (already partially done for GTHA)
