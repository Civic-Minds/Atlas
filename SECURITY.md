# Security Policy

## Supported Versions

Only the latest version of Atlas is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.8.x   | :white_check_mark: |
| < 0.8.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it via [GitHub Issues](https://github.com/Civic-Minds/Atlas/issues) or contact the maintainer directly.
